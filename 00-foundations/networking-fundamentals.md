# Networking Fundamentals

Core networking concepts required before diving into AWS VPC and connectivity services.

## Key Topics

- **OSI Model** — understand layers 1–7 and how each maps to AWS services (e.g., ALB = L7, NLB = L4)
- **IP Addressing & CIDR** — IPv4/IPv6 notation, subnetting, calculating usable host ranges for VPC design
- **Routing** — default gateways, route tables, BGP basics relevant to Direct Connect and VPN
- **DNS** — how resolution works end-to-end; TTL, A/AAAA/CNAME/MX/TXT records, split-horizon DNS
- **Firewalls & NAT** — stateful vs. stateless packet filtering; NAT Gateway vs. NAT instance tradeoffs
