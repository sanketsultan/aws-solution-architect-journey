# Linux Basics

Essential Linux/Unix skills for working with EC2 instances, containers, and CLI tooling.

## Key Topics

- **File System & Permissions** — `chmod`, `chown`, directory structure (`/etc`, `/var`, `/tmp`), inodes
- **Process Management** — `systemd`, `ps`, `top`, signals (`kill`, `SIGTERM`, `SIGKILL`), background jobs
- **Networking Commands** — `curl`, `netstat`/`ss`, `ping`, `traceroute`, `dig`, `iptables` basics
- **Shell Scripting** — bash variables, loops, conditionals, exit codes, piping and redirection
- **Package Management** — `apt`/`yum`/`dnf`, installing the AWS CLI, common troubleshooting tools
