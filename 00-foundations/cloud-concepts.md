# Cloud Concepts

Foundational cloud computing concepts that underpin all AWS services.

## Key Topics

- **Cloud Service Models** — IaaS, PaaS, SaaS; knowing which AWS services fall into each category
- **Deployment Models** — public cloud, private cloud, hybrid cloud, multi-cloud tradeoffs
- **CapEx vs. OpEx** — shifting from capital expenditure to operational expenditure with cloud adoption
- **Elasticity & Scalability** — horizontal vs. vertical scaling; Auto Scaling Groups; on-demand capacity
- **Shared Responsibility Model** — what AWS owns (security *of* the cloud) vs. what you own (security *in* the cloud)
