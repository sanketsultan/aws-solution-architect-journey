# Amazon CloudWatch

Observability service — metrics, logs, alarms, dashboards, and automated actions.

## Key Topics

- **Metrics & Namespaces** — built-in AWS metrics; custom metrics via `PutMetricData` API; high-resolution metrics (1 s)
- **CloudWatch Logs** — Log Groups, Log Streams, metric filters to extract metrics from logs; Logs Insights for querying
- **Alarms** — threshold-based or anomaly detection; states: OK, ALARM, INSUFFICIENT_DATA; actions: SNS, Auto Scaling, EC2
- **Container & Application Insights** — Container Insights for ECS/EKS; Application Insights for .NET and SQL Server apps
- **CloudWatch Agent** — collect OS-level metrics (disk, memory) and logs from EC2 and on-premises servers
