# AWS X-Ray

Distributed tracing for analyzing and debugging microservices and serverless applications.

## Key Topics

- **Traces & Segments** — a trace = end-to-end request; segments = work done by one service; subsegments = granular breakdown
- **Service Map** — visual graph of your application's services and their connections; shows latency and error rates
- **Sampling** — configurable sampling rules to control trace volume; reservoir + fixed rate; reduce cost without losing coverage
- **X-Ray SDK & Daemon** — SDK instruments your code; daemon buffers and forwards segments to X-Ray API
- **Integration** — native support in Lambda, API Gateway, ECS, EC2 (via daemon), Elastic Beanstalk, App Mesh
