# Amazon Aurora

AWS-built relational database — MySQL and PostgreSQL compatible, cloud-native storage engine.

## Key Topics

- **Aurora Architecture** — shared distributed storage volume (6 copies across 3 AZs); compute and storage decoupled
- **Aurora vs. RDS** — up to 5x faster than MySQL RDS; automatic storage growth (10 GB → 128 TB); faster failover (~30 s)
- **Aurora Replicas** — up to 15 read replicas; failover target; Auto Scaling for reader fleet
- **Aurora Serverless v2** — instant scaling in fine-grained ACU increments; good for variable/unpredictable workloads
- **Aurora Global Database** — one primary region + up to 5 secondary regions; < 1 s replication lag; managed failover
