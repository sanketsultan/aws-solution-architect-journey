# AWS WAF & AWS Shield

Web application firewall and DDoS protection.

## Key Topics

- **WAF Web ACLs** — rules applied to CloudFront, ALB, API Gateway, AppSync; managed rule groups (OWASP Top 10, bot control, AWSManagedRulesCommonRuleSet)
- **WAF Rule Types** — IP sets, geo-match, rate-based rules, regex pattern sets, SQL injection/XSS match
- **Shield Standard** — free, automatic L3/L4 DDoS protection for all AWS resources
- **Shield Advanced** — paid; L7 protection, near real-time attack visibility, DDoS cost protection, 24/7 DRT access
- **Firewall Manager** — centrally deploy WAF rules, Shield, SG policies, and Network Firewall across an AWS Organization
