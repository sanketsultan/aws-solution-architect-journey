# Amazon CloudFront

Global Content Delivery Network (CDN) with 400+ edge locations.

## Key Topics

- **Origins & Behaviors** — S3 origins, custom (HTTP) origins, origin groups for failover; cache behavior path patterns
- **Cache Control** — TTL settings, Cache-Control headers, invalidation (cost and latency); cache key customization
- **Security** — HTTPS enforcement, custom SSL certificates (ACM in us-east-1), field-level encryption, signed URLs & signed cookies
- **Origin Access Control (OAC)** — replacing legacy OAI; restricting S3 bucket to CloudFront only
- **Lambda@Edge & CloudFront Functions** — request/response manipulation at edge; use-case comparison (latency, language, event type)
