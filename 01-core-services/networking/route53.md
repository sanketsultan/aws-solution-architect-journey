# Amazon Route 53

Scalable, highly available DNS and domain registration service.

## Key Topics

- **Routing Policies** — Simple, Weighted, Latency-based, Failover, Geolocation, Geoproximity (Traffic Flow), Multi-Value Answer, IP-based
- **Health Checks** — endpoint health checks, calculated health checks, CloudWatch alarm checks; used with failover routing
- **Private Hosted Zones** — DNS resolution within a VPC; associating multiple VPCs; split-horizon DNS
- **Alias Records** — Route 53 extension to DNS; point to AWS resources (ELB, CloudFront, S3, API GW) with no TTL charge
- **DNSSEC** — signing hosted zones; DS record delegation to registrar
