# Amazon S3 Glacier

Long-term archival storage — lowest cost, higher retrieval latency.

## Key Topics

- **Glacier Storage Tiers** — Glacier Instant Retrieval (ms), Glacier Flexible Retrieval (minutes–hours), Glacier Deep Archive (hours)
- **Retrieval Options** — Expedited, Standard, Bulk; cost and time tradeoffs per tier
- **Vault Lock vs. Object Lock** — Vault Lock (legacy Glacier API, compliance mode via vault access policy); Object Lock (S3-native WORM)
- **Lifecycle Policies** — transitioning objects from S3 Standard → IA → Glacier automatically based on age
- **Use Cases** — regulatory compliance archives, media asset preservation, database backup retention
