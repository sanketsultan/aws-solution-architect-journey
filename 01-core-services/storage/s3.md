# Amazon S3 — Simple Storage Service

Object storage — infinitely scalable, highly durable (11 nines) storage.

## Key Topics

- **Storage Classes** — Standard, Intelligent-Tiering, Standard-IA, One Zone-IA, Glacier Instant/Flexible/Deep Archive; lifecycle policies
- **Bucket Policies & ACLs** — resource-based policies, Block Public Access settings, ACL deprecation path
- **Versioning, MFA Delete & Object Lock** — protecting data from accidental deletion; WORM compliance with Object Lock
- **Encryption** — SSE-S3, SSE-KMS (audit trail), SSE-C (customer-managed keys), client-side encryption
- **Performance & Transfer** — multipart upload (>100 MB), S3 Transfer Acceleration, byte-range fetches, strong consistency model
